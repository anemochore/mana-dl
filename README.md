# mana-dl
userscript downloaders for manatoki, 11toon, etc

## 사용법
1. 먼저 탬퍼멍키를 설치한다. Tampermonkey는 공홈 가서 자기 브라우저에 맞게 받자. https://www.tampermonkey.net/
2. Tampermonkey를 깔았다면 대시보드 가서 새 스크립트 만들고 전체 내용을 지우고, 마나토끼는 [`manatoki-dl-tm-gh.js`](https://anemochore.github.io/mana-dl/manatoki-dl-tm-gh.js), 일일툰은 [`11toon-dl-tm-gh.js`](https://anemochore.github.io/mana-dl/11toon-dl-tm-gh.js) 파일의 내용으로 전체를 교체한다.
3. 그다음부터는 해당 사이트에서 화수 표시된 목록 페이지 들어가면 체크박스와 다운로드 버튼이 자동으로 생김. 받을 화 체크하거나 입력하고 다운로드 버튼을 누르면 되는데, 스크립트 최초 실행 및 다운로드 시작 시에 다른 사이트 접근을 허용할 건지 물어보면 허용한다고 해야 함.
4. 각 파일 코드 맨 앞에 사이트별로 조금 더 자세한 주의 사항이 써 있음.
